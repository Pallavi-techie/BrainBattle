# 🧠 BrainBattle

Gamified Study & Multiplayer Quiz Platform for School Students

---

## 🚀 Overview

BrainBattle is a full-stack web platform designed for school students to study together and compete through real-time quiz battles.

It combines:

* 📚 Study Circle (private academic space)
* 🎮 Fun Arena (multiplayer quiz battles)
* 🏆 Gamification (coins, badges, streaks)

---

## ⚙️ Tech Stack

* Frontend: React.js + Tailwind CSS
* Backend: Node.js + Express.js
* Database: MongoDB
* Real-Time: Socket.IO
* Auth: JWT

---

## 🔥 Key Features

* Real-time multiplayer quiz engine
* Study circles with homework + flashcards
* Coins, badges, leaderboard system
* Admin panel for quiz management

---

## 📂 Project Structure

* brainbattle-frontend → React app
* brainbattle-backend → API + Socket.IO server
* docs → Project documentation

---

## 🌍 Vision

To create a focused, fun, and inclusive study platform for students across different schools, boards, and languages.
